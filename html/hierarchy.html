Иерархия классов.
[уровень детализации 12]
 ▼Cgame.Player.BuildingКласс Building даёт возможность строительства игроку. Так же предоставляет возможность улучшать здания
 Cgame.Player.Buildings.DrillКласс Drill служит для добычи железа в замен на энергию
 Cgame.Player.Buildings.FactoryКласс Factory служит для производства компонентов разного уровня в замен на железо и энергию
 Cgame.Player.Buildings.GamegoalКласс Gamegoal служит целью для игрока. После постройки, игра считается пройденной
 Cgame.Player.Buildings.SandQuarryКласс SandQuarry служит для добычи песка в замен на энергию
 Cgame.Player.Buildings.SteamEngineКласс SteamEngine служит для переработки воды в энергию
 Cgame.Player.Buildings.WarehouseКласс Warehouse служит для склада ресурсов. Его постройка и улучшения увеличивают вместимость ресурсов
 Cgame.Player.PumpКласс Pump служит для добычи воды в замен на энергию
 Cgame.World_map.ChunkКласс, хранящий в себе изображение блоков. Так же хранит в себе типы блоков
 ▼CForm
 Cgame.FormGameОсновное окно игры. Служит для отображения игрового поля, зданий, ресурсов
 Cgame.StartMenuСтартовое окно игры. Даёт выбор пользователю выйти из игры, загрузить сохранение или начать новую игру
 Cgame.World_map.MapКласс Map служит созданию игровой области. Хранит в себе размер карты, игровые области(чанки)
 Cgame.Player.PlayerКласс Player хранит в себе информация о ресурсах. В хранение входит: кол-во каждого ресурса, кол-во выробатывания его или поглощения
 Cgame.ResourcesInfoКласс для вывода информации по ресурсам. Выводит кол-во ресурсов, их увеличение или уменьшение
 ▼Cgame.World_map.SpritesКласс Sprites хранит в себе параметры изображения. Он задаёт размер блоков, зданий, кол-во пикселей в блоке
 Cgame.World_map.Block.GrassКласс Grass Пресдавляет собой шаблон для создания блока типа Grass, с соответсвующими цветами
 Cgame.World_map.Block.OreКласс Ore Пресдавляет собой шаблон для создания блока типа Ore, с соответсвующими цветами
 Cgame.World_map.Block.SandКласс Sand Пресдавляет собой шаблон для создания блока типа Sand, с соответсвующими цветами
 Cgame.World_map.Block.WaterКласс Water Пресдавляет собой шаблон для создания блока типа Water, с соответсвующими цветами
